Mulai Cepat — 5 menit
17 Mei 2026 · 3 menit baca

Kagi menghadirkan pencarian tanpa iklan dan sangat privat. Untuk memulai, atur sebagai mesin pencari default di browser.

Langkah awal

1. Pasang ekstensi Kagi di Chrome/Firefox.
2. Sesuaikan preferensi (naikkan atau turunkan domain).
3. Coba fitur !bang seperti !yt atau !github.

Contoh kode highlighter dengan JavaScript:

    console.log("Hello, " + name + "!");

Cara Cari di Kagi
17 Mei 2026 · 4 menit baca

Personalisasi hasil pencarian: naikkan situs favorit atau blokir domain langsung dari hasil pencarian.

Tips pencarian

Gunakan tanda tanya ? di akhir kata kunci untuk memicu AI Quick Answer. Contoh: resep nasi goreng? akan memberi ringkasan.

Lensa juga bisa memfilter topik dari forum atau situs akademik.

# Python example
def search(query):
    return f"Hasil untuk {query}"
print(search("kagi"))
Lensa dan Shortcut (Bang)
17 Mei 2026 · 5 menit baca

Lensa adalah filter pintar yang menyaring website tertentu. Contoh: Lensa "Programming" hanya menampilkan hasil coding.

!bang shortcut ajaib: !yt lofi hip hop (YouTube), !mdn flexbox (MDN), !gh kagi-docs (GitHub).

Assistant AI — Tanpa Sampah
17 Mei 2026 · 4 menit baca

Kagi Assistant menggabungkan GPT-4, Claude, Llama tanpa melacak privasi. Bisa merangkum halaman atau menjawab pertanyaan.

Fitur andalan

Tekan Summarize Page untuk meringkas artikel. Gunakan "Ask a question" terhadap URL tertentu. Semua tanpa iklan.

Privasi Bawaan dan API Kuat
17 Mei 2026 · 6 menit baca

Kagi tidak menyimpan riwayat pencarian, tanpa pelacak, 100% bebas iklan. Untuk developer tersedia API Kagi Search.

Dokumentasi teknis

Integrasikan Kagi ke aplikasi, bot, atau workflow. API mencakup pencarian real-time, summarizer, lensa kustom. Lihat help.kagi.com/api.

curl -X GET "https://api.kagi.com/search?q=example"
